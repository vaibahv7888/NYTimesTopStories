# NYTimesTopStories
New York Times top stories reader using the nytimes public API.

## The project is implemented using following principals:
- Clean and Layered Architecture.
- Viper Design Pattern.
- SOLID principals.

## Implementation Details:
- No external dependencies or package managers are used
- Test cases are added.
- Separation of concerns and organised code.

## System Requirements:
- Swift 5
- Xcode 11

## Note:
The project contains Coredata stack but it is not used as it was not necessary for the given task and due to time contrained.
